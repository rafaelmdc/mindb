Microbiome Association Database

A small Django-based database for curated microbiome study data, focused on organism-organism associations, structured sample metadata, and clear internal browsing workflows.

The project centralizes evidence from published studies so the team can review studies, browse cohorts and samples, inspect organism records, and trace pairwise associations without relying on spreadsheets alone.

What the database contains

Study-level metadata, sample and cohort context, organism records, pairwise relative associations, and controlled flexible metadata for variables that differ across papers.

Why it matters

Microbiome interactions are easier to reason about when the underlying evidence is normalized, searchable, and tied back to the study and sample context where each association was reported.

Current workflow

Use the browser for inspection, Django admin for manual curation, and the import workflow for validated batch ingestion. The graph view is available as an exploratory layer over curated associations.
