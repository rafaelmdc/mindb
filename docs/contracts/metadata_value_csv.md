# Metadata Value CSV

Required columns:

- `study_doi`
- `study_title`
- `group_name`
- `variable_name`

At least one of `study_doi` or `study_title` must be populated in each row.

Typed value columns:

- `value_float`
- `value_int`
- `value_text`
- `value_bool`

Exactly one typed value column may be populated.
